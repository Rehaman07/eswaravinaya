# 🍽️ Eswar Vinayaka Hotel Website

A modern and visually engaging restaurant website showcasing authentic South Indian tiffins, meals, and homestyle hospitality. Designed to attract customers with a smooth and inviting user experience.

---

## 🌐 Live Demo

👉 https://eswaravinaya123.vercel.app/

---

## ✨ Features

* 🍛 Beautiful hero section with strong branding
* 📋 Menu preview (Tiffins, Meals, Snacks, Sweets)
* ⭐ Customer reviews section
* 📍 Location & visit planning
* 📱 Fully responsive design (mobile-friendly)
* ⚡ Fast performance and smooth UI

---

## 🎨 UI/UX Highlights

* Clean and modern layout
* Warm South Indian themed color palette
* Smooth navigation experience
* Clear call-to-action buttons
* Designed for real customer engagement

---

## 🚀 Getting Started

### 1. Clone the repository

```
git clone https://github.com/Rehaman07/eswervinaya.git
```

### 2. Navigate to project

```
cd eswervinaya
```

### 3. Install dependencies

```
npm install
```

### 4. Run development server

```
npm run dev
```

---

## 📦 Build for Production

```
npm run build
```
